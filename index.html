Sobre

Oi! Meu nome é Rodrigo Pinilla Hamada, seja Bem-Vindo(a)!

Iniciei no universo da tecnologia com cerca de 16 anos de idade. Meu primeiro projeto foi a elaboração de um site criado a partir de HTML para a criação de uma web rádio. Desenvolvi o site utilizando o Macromedia Dreamweaver 8, estudando por conta própria. Desde então, sou apaixonado pela área. Desse modo, fui interagindo com diversas aplicações, tais como Adobe Photoshop, Sony Vegas, Code Blocks, Visual Studio Code etc, a fim de aperfeiçoar os meus conhecimentos. Gosto de atuar na parte gráfica, bem como em sua codificação. Atualmente, estou cursando ensino superior em Análise e Desenvolvimento de Sistemas, estudando linguagens de programação e novas tecnologias. Tive uma passagem breve como estagiário de TI na Prefeitura Municipal de Valinhos, atuando em suporte e infraestrutura. Possuo entendimento em linguagem C e lógica de programação. Busco maiores conhecimentos em Python, C#/C++, Java, Javascript. Além disso, estou estudando aplicações web, praticando SQL e suporte técnico em TI, como também, conhecimentos em pacote Office e Libreoffice.
